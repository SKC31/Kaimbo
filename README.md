# Personal Portfolio Website

## Overview

This project is my personal portfolio website. It was created to present my background, skills, qualifications, and experience as a software engineering student.

The goal of this project is to provide a simple and professional platform where I can showcase my work and introduce myself to potential collaborators, recruiters, or anyone interested in my work.

## Features

* Sidebar navigation for quick access to sections
* Smooth scrolling between sections
* Typewriter-style introduction animation
* Responsive layout for different screen sizes
* Skills section with scrollable cards
* Contact information for communication

## Technologies Used

* HTML
* CSS
* JavaScript

## How to Run

1. Download or clone the repository from GitHub.
2. Open the project folder.
3. Open Portfolio_update.html in a web browser.

No additional setup is required.
